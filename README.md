### Data Structures

  - Bubble Sort, Selection Sort, Insertion Sort

  - Breadth-First Search (BFS)
  
- **Graph Algorithms**
  - Dijkstra's Algorithm
  - Bellman-Ford Algorithm
  - Floyd-Warshall Algorithm
  - Kruskal's and Prim's MST Algorithms
  
- **Dynamic Programming**
  - Fibonacci Sequence
  - Knapsack Problem
  - Longest Common Subsequence
  - Edit Distance
